# GuilisDasaAHK

Scripts AutoHotkey v2 para geração de laudos anatomopatológicos estruturados.

## Descrição

Ferramenta de produtividade para patologistas. Abre GUIs interativas (máscaras) que coletam os achados do exame e geram automaticamente o texto do laudo formatado, reduzindo tempo de digitação e padronizando os resultados.

## Órgãos/sistemas cobertos

- **Esôfago** — biópsia (esofagite, tumor) e peça cirúrgica
- **Estômago** — normal, biópsia (gastrite, tumor) e peça cirúrgica
- **Duodeno** — normal, duodenite, doença celíaca
- **Apêndice** — normal, tumor, NET
- **Cólon** — biópsia, tumor (pediculado, séssil, peça), NET, colite crônica
- **Intestino delgado** — normal, NET (jejuno/íleo), GDP
- **Pele e Dermatopatologia** — CBC, CEC, melanoma, nevo, queratose, lúpus, hanseníase, alopecia, dermatite, hiperplasia epidérmica
- **Ginecológico** — colo uterino (biópsia, NIC, tumor), endométrio, ovário, tuba uterina
- **Pulmão** — biópsia e peça cirúrgica
- **Rim** — nefrectomia parcial/total, exclusão, tumor de Wilms
- **Próstata** — benigno, biópsia (tumor), RTU, peça cirúrgica
- **Mama** — benigno, biópsia, filoide, mastectomia, protocolo QT, IHQ
- **Sarcomas**
- **Estadiamento** — estômago, cólon, esôfago, rim, mama

## Requisitos

- [AutoHotkey v2](https://www.autohotkey.com/) instalado
- Windows 10/11

## Uso

Execute `Guilis.ahk` com o AutoHotkey v2. O ícone aparece na bandeja do sistema — clique para acessar o menu de máscaras.
